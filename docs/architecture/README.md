# Architektur — Zweck & Inhalte

Dieser Ordner enthält technische Spezifikationen und Architektur‑Dokumente für das Projekt.

Wichtige Dateien

- `api.md` — API‑Endpunkte, Auth, Fehlerformate
- `database-schema.md` — PostGIS‑Datenmodell, Indizes, Migrationsempfehlungen
- `mapstack.md` — Entscheidung zum Mapstack (extern) und Anforderungen

Richtlinien
