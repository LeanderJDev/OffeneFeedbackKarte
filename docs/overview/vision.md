# Vision

Eine Online-Plattform auf der man über eine Karte Pins sehen und setzen kann, um Hinweise, Rückmeldungen oder Verbesserungsvorschläge anzugeben. Diese Pins können dann von anderen Nutzer:innen kommentiert und bewertet werden. Zusätzlich können zuständige Stellen auf die Pins reagieren und gegebenenfalls Maßnahmen ergreifen.
