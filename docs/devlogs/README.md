# Devlogs

In `devlogs/` werden kurze Protokolle über Entwicklungs‑Sessions, Experimente und besondere Änderungen abgelegt.

Konventionen

- Dateiname: `YYMMDD[-suffix].md` (z. B. `260206-ai-assisted.md`).
- Inhalt: Datum, Ziel der Session, durchgeführte Änderungen und next steps.
- Keine ausführliche Spezifikation: Devlogs sind chronologisch und kurz.

Beispiel

- 2026-02-06 — AI‑unterstützte Überführung von Notizen in `docs/` (Kurzbeschreibung, Dateiliste, Entscheidungen).
